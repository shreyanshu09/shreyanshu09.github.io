LayGen

Award Winner

NEOALI Co. Ltd. · 2025–Present

Live Product ↗

LayGen is a multi-platform AI document translation ecosystem that preserves original document formatting across DOCX, HWPX, PDF, PPTX, and XLSX. The system integrates on-device LLM inference (via Ollama) for enterprise-grade offline security, eliminating the need to send sensitive documents to cloud servers. A desktop application (ElectronJS) and server-side API (FastAPI + Python) provide a seamless end-to-end user experience. Won 1st Place at Intel AI PC Innovation Challenge 2024.
